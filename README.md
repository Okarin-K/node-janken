# Node.js で作ったじゃんけんゲーム

## 動作環境

Node.js v16.13.0

## 遊び方

1. npm のパッケージをインストールしてください。

```bash
npm install
```

2. npm scripts の build tsc を実行。  
   または、以下のコマンドを実行してビルドしてください

```bash
npx tsc --build
```

3. npm scripts の run を実行。  
   または、以下のコマンドを実行してサーバーを起動させてください。

```bash
node ./dist/app.js
```

4. サイトの URL にアクセスして、自由に遊んでください

    http://localhost:5000/
